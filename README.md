# Google - Site Imitação

Este projeto é uma recriação simples da página inicial do Google, feita utilizando apenas HTML e CSS. O objetivo desse projeto foi criar uma versão semelhante da interface do Google, com um design minimalista e funcional.

Funcionalidades
Página Inicial: O site apresenta uma interface similar à do Google, com um campo de pesquisa no centro da página, o logo do Google e os links habituais (como "Sobre", "Privacidade", etc).

Layout Simples e Limpo: O design segue a estética minimalista característica do Google, com ênfase em uma experiência de usuário limpa e focada na busca.

Tecnologias Utilizadas
HTML: Estrutura e conteúdo do site.
CSS: Estilização e layout responsivo da página.
